AI Resume Analyzer 📄🤖

AI Resume Analyzer is a smart web application that helps users improve their resumes by comparing them with job descriptions using Artificial Intelligence and Machine Learning.

It provides resume match score, ATS suggestions, missing skills analysis, and professional recommendations to increase job selection chances.

---

🚀 Features

✅ Upload Resume in PDF format
✅ Compare Resume with Job Description
✅ Resume Match Score (%)
✅ AI Suggestions to Improve Resume
✅ ATS Optimization Tips
✅ Missing Skills Detection
✅ Download Analysis Report

---

🛠️ Tech Stack

- Python
- Streamlit
- Scikit-learn
- NLP (NLTK)
- Groq API
- PyPDF2 / pdfplumber
- Matplotlib
